***Release Notes 0.62***

This service is running in operating form here: http://www.saveofficedata.com. As far as I know this is a Microsoft Windows server only script because of the use of multiple Windows command line apps.

I'm still looking for a good Open Source uploader with a progress bar in which to include this script.  Any way here it is with the proprietary uploader that I use' code removed. Hopefully I got rid of it all.  You will need to find your own file uploader somehow point it to my klugee code :-|.  I use the supload uploader and recommend it for it's ease of use.  

I set my code down in the middle of the machinry after it has created the scalar variable $success[0], which represents an uploaded file.  I also strip out the extra fileds in the client so that only one file is uploadable so there will be no $success[1], $success[2] etc.  It's here:

http://www.perlservices.net/en/programs/psupload/

**Installation - Alpha Attempt**

1. Download and unzip in your home directory.
2.  Move upload.cgi to the cgi direcotry.
3.  Make sure that upload.cgi is the equivalent of Linux 755 permisions (maybe the directory too).
4. Find a php, perl or python uploader.
5. Have it upload files to the cgi bin (sorry this might be a stupid iunsecure move - I'm open to suggestions).
6. Once the file uploader has done it's work, have it point to the upload.cgi script and make sure it has named the upload file the equivalent of "$success[0]" or change this to you own variable (which is a better idea I know...I'm really a beginner...).
7. Download all the executable and unzip their parent zip files in the cgi-bin.
